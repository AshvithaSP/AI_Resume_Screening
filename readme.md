A web-based AI-powered Resume Screening and Applicant Tracking System (ATS) developed using Python and Streamlit. The application intelligently analyzes resumes, extracts technical skills, evaluates candidate suitability, recommends job roles, identifies skill gaps, and ranks multiple candidates for recruiters through an interactive dashboard.

## 📌 Project Overview

**SmartHire AI** is an AI-powered Resume Screening System that assists both **job seekers** and **recruiters**.

It automatically analyzes resumes, extracts technical skills, calculates ATS scores, recommends suitable job roles, identifies skill gaps, and ranks multiple candidates for recruiters.

The system provides two modes:

- 👨‍🎓 Candidate Analysis
- 👨‍💼 Recruiter ATS Dashboard

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/ASHVITHASP/SmartHire-AI.git
```

Go into the project

```bash
cd SmartHire-AI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app_combined.py
```
